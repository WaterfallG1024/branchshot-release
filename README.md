# BranchShot

スクショを、もう散らかさない。
Windows向けのスクリーンショットアプリです。

**[公式サイト（使い方・ダウンロードはこちら）](https://waterfallg1024.github.io/branchshot-web/)**

## 概要
BranchShotは、フォルダ単位で簡単にスクリーンショットを整理でき、また最大1分前までの画面をスクリーンショットできるアプリケーションです。

## ダウンロードとインストール
最新版のダウンロードおよび詳しい使い方は、公式サイトをご覧ください。
https://waterfallg1024.github.io/branchshot-web/

## 動作環境
- **OS**: Windows 10 (バージョン2004以降) / Windows 11
- **CPU**: x64 互換プロセッサ
- **メモリ**: 4GB 以上 (タイムトラベル機能には 8GB以上必須)
- **グラフィックス**: DirectX 9 互換 GPU (タイムトラベル機能には DirectX 11 互換推奨)
